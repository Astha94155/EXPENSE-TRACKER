# Expense Tracker 

## Problem Statement

Managing day-to-day expenses can be difficult for users who need a quick and simple way to record what they spend. Many people want a minimal tool without complex features, where they can directly store purchases and amounts. This project provides a basic console-based solution for tracking expenses efficiently.

## Scope of the Project

* Focuses on creating a simple, user-friendly expense tracking system.
* Only basic functionalities such as adding and viewing expenses are included.
* The project is intended for beginners learning Python loops, lists, and input handling.
* Does not include advanced analytics, graphs, or data storage in files/databases.

## Target Users

* Students learning Python programming.
* Beginners who want to build their first console project.
* Anyone who needs a very simple personal expense tracker without extra complexity.

## High-Level Features

* **Add Expense:** Users can enter an item's name and amount spent.
* **View All Expenses:** Displays the entire expense list in an organized manner.
* **Exit Function:** Allows users to exit the program at any time.
* **Continuous Operation:** Program runs in a loop until the user decides to quit.
