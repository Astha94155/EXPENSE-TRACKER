# Expense Tracker

## Overview of the Project

This is a simple Python-based console program that allows users to record and review their daily expenses. The user can add items they purchased along with the amount spent and can view all recorded expenses at any time.

## Features

* Add an expense (item name + amount spent).
* View all stored expenses in a list format.
* Exit the program anytime.

## Technologies/Tools Used

* **Python 3**
* **NumPy** (imported but not required for functionality)
* Basic Python data structures like lists and loops.

## Instructions for Testing

1. Run the program.
2. Enter:

   * **1** to add a new expense.
   * **2** to view the list of all added expenses.
   * **3** to exit the program.
3. Test by adding different items and amounts.
4. Verify that all added items display correctly when selecting option **2**.
